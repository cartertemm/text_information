# Текстовая информация

Это дополнение предоставляет пользователям контекстную информацию, подходящую для широкого спектра случаев использования.
Одним нажатием клавиши оно может сообщить значение слова, определить местоположение IP-адреса и рассказать о книге (по ISBN). Просто выделите что-нибудь, используйте назначенную комбинацию клавиш и подождите.

## Поддерживаемые сервисы

В настоящее время поддерживаются следующие функции:

* Информация об IP-адресе. Включает геолокацию, провайдера (ISP), определение выходных узлов VPN/Tor и сотовых сетей.
* Определения из английского словаря, часть речи, примеры предложений, синонимы, антонимы и т. д. Предоставлено [Free Dictionary API](https://dictionaryapi.dev/). При отображении определения слова в диалоговом окне с возможностью просмотра включены кнопки для прослушивания произношения, если аудио доступно
* Поиск по ISBN через API Google Books
* Информация об URL. Получает заголовок страницы, метаописание, размер содержимого и конечный домен (если URL перенаправляет на другой домен), прежде чем вы перейдёте на сайт.
* Проверка типа кредитной карты (Mastercard, Visa, Discover, Amex и т. д.)

Дополнение также поддерживает распознавание номеров телефонов и адресов электронной почты, хотя никакая дополнительная информация о них не получается. Это может измениться, как только я найду, что с ними делать, и подходящий API, соответствующий нашим требованиям.

Примечание: для проверки данных используются регулярные выражения. Это означает, что адреса электронной почты и номера карт никогда не покинут ваше устройство.

## Клавиши

Примечание: эти комбинации клавиш рассчитаны на английскую раскладку клавиатуры и могут не работать при другой раскладке. Если у вас возникла проблема, сначала попробуйте изменить их в диалоговом окне жестов ввода.

* NVDA+; (точка с запятой): предоставляет информацию на основе выделенного текста
* NVDA+SHIFT+; (точка с запятой): предоставляет информацию о тексте в буфере обмена
* NVDA+control+; (точка с запятой): произносит последнюю полученную информацию. Быстро нажмите дважды, чтобы отобразить её в диалоговом окне с возможностью просмотра. Для определений слов это диалоговое окно также включает кнопки для воспроизведения аудио произношения, если оно доступно. Escape закрывает диалоговое окно.

## Примечание о Python 3

Начиная с версии NVDA 2019.3, все дополнения должны быть совместимы с Python 3. Если по какой-то причине вы используете более старую версию, [1.0](https://github.com/cartertemm/text_information/releases/download/1.0/textInformation-1.0.nvda-addon) — последняя версия, работающая с Python 2, причём определения из словаря в ней больше не работают из-за прекращения поддержки Princeton Wordnetweb. Обе версии следует считать неподдерживаемыми.
